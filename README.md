# 📊 Retail Store KPI Dashboard

An interactive Retail Store KPI Dashboard built in Microsoft Excel to analyze sales performance, profitability, customer behavior, and product trends using data visualization and business intelligence techniques.

This project demonstrates practical data analytics skills including dashboard creation, KPI tracking, Pivot Tables, Pivot Charts, slicers, conditional formatting, and business insight generation.

---

# 🚀 Project Overview

The goal of this project was to transform raw retail sales data into an interactive business dashboard capable of helping stakeholders monitor:

- Sales performance
- Profitability
- Product trends
- Regional performance
- Customer segments
- Business KPIs

The dashboard enables dynamic filtering and visual analysis to support data-driven decision-making.

---

# 🎯 Objectives

- Analyze retail sales data using Excel
- Build an interactive KPI dashboard
- Identify high-performing regions and products
- Track profit margins and order trends
- Generate business insights from raw data
- Create a professional portfolio-ready analytics project
